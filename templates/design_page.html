🧭 Behind the Resume

This isn't a drag-and-drop Wix site. It's a hand-crafted, containerized web app orchestrated with love, logic, and real infrastructure. Here's how it works:

🧱 Architecture

  • Stack: Docker Compose powering Django + Postgres + pgAdmin
  • Modular Design: Each service lives in its own folder with clear separation
  • Environment Config: A shared .env file centralizes config
  • Database: Initialized with init.sql, pgAdmin auto-login enabled

🗂️ Folder Structure

resume-app/
├── backend/          # Django project with modular apps
│   ├── resume/       # Core logic for resume content
│   ├── static/       # CSS and JS assets
│   └── templates/    # HTML templates
├── db/               # Postgres init script
│   └── init.sql
├── docker-compose.yml
├── .env
└── README.md         # Full setup and dev instructions

⚙️ Key Features

  • Dynamic rendering of resume data using Django templates
  • Modular index page for future expansion
  • pgAdmin pre-configured for seamless database visibility
  • Deployed via Heroku using container workflows
  • Backed by version control on GitHub

🧪 Built with Resilience

  • Stack rebuilt and debugged multiple times for clarity and fault tolerance
  • Documented every layer, from .env to init.sql
  • Visual debugging, screenshot audits, and deep reproducibility
